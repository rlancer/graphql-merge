# GraphQL - Merge Example 

> An example which shows how to merge 2 data sources into one via GraphQL

## Getting started 

```bash
git clone git@github.com:rlancer/graphql-merge.git
cd graphql-merge 

# yarn  
yarn
yarn start 

# npm 
npm i 
npm start
```
